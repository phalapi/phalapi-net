快速使用 PhalApi

  • 1、进入Github后从release分支下载，或直接：
    git clone https://github.com/phalapi/phalapi.git
  • 2、上传并解压到服务器 （推荐使用nginx，并将根目录设置为Public）
  • 3、打开浏览器，访问安装向导：
    http://localhost/PhalApi/Public/install/
  • 4、最后运行效果：
    http://localhost/PhalApi/Public/demo/
