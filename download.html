PhalApi框架下载

推荐从GIT@OSC（国内通道）下载最新代码：

git clone https://git.oschina.net/dogstar/PhalApi.git

或从Github（国际通道）下载：

git clone https://github.com/phalapi/phalapi.git

PhalApi-Library扩展类库下载

推荐从GIT@OSC下载最新代码：

git clone https://git.oschina.net/dogstar/PhalApi-Library.git

或

git clone https://github.com/phalapi/phalapi-library.git

Wiki文档下载 - HTML格式（从markdown生成，可部署在本地或内部网络）

git clone https://git.oschina.net/dogstar/PhalApi-Net.git
cd wikis

cd PhalApi.wiki
