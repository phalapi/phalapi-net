\PhalApi_Api

PhalApi_Api 接口服务基类

  • 实现身份验证、按参数规则解析生成接口参数等操作
  • 提供给开发人员自宝义的接口服务具体类继承

通常地，可以这样继承：

class Api_Demo extends PhalApi_Api {

 public function getRules() {
     return array(
         // ...
     );
 }

 public function doSth() {
     $rs = array();

     // ...

     return $rs;
 }

}
